📌 NOTES

  • Pipeline runs offline — processes the full Jira CSV export in a single batch (no incremental sync)
  • LLM calls are cached to disk after completion — restarts skip expensive API calls instantly
  • All outputs are versioned with generation timestamps and provenance metadata
  • Designed for scalability — can process thousands of issues with batched LLM calls
  • Overall LLM Strategy: Use LLM only for semantic understanding (extraction + validation); use deterministic rules for structure, parsing & graph construction
